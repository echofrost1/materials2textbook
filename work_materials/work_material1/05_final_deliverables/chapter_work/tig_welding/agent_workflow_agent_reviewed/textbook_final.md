# 第X章 钨极氩弧焊（TIG）基础与操作

## 1.1 设备组成与系统认知

### 学习目标
1. 能正确识别手工钨极氩弧焊设备的12个主要组成部分。
2. 能复述弧焊电源外特性的基本概念及电焊机铭牌参数的含义。
3. 能说明TIG焊程序控制系统中气、水、电的同步逻辑及安全要求。
4. 能列举焊接回转台与变位机等辅助设备的功能与应用场景。

### 知识讲解

#### 1.1.1 手工钨极氩弧焊设备的组成
手工钨极氩弧焊设备是一个由多个子系统组成的复杂整体。根据标准配置，其主要由以下12个部分组成：
1. **焊接电源及控制系统**：提供焊接所需的电能并控制焊接过程。
2. **供气系统**：提供保护气体（如氩气）。
3. **供水系统**：用于水冷式焊枪的冷却。
4. **焊枪**：夹持钨极、输送保护气体和冷却水，并传导焊接电流。
5. **焊丝**：作为填充金属。
6. **工件**：被焊接的对象（注：在系统连接示意图中，工件作为回路的一部分被列出）。
7. **工件电缆**：连接工件与电源负极（或正极）。
8. **焊枪电缆**：连接焊枪与电源。
9. **出水管**：排出冷却水。
10. **开关线**：控制焊枪开关信号。
11. **焊枪气管**：输送保护气体至焊枪。
12. **供气气管**：连接气源与焊枪。
`证据：PPT_A000145_S010`

#### 1.1.2 弧焊电源特性与铭牌参数
**弧焊电源外特性**是指在电弧稳定燃烧状态下，弧焊电源输出电压与输出电流之间的关系，通常用外特性曲线表示。弧焊电源的外特性主要分为三种基本类型：
* 下降外特性
* 平外特性
* 上升外特性
`证据：PPT_A000056_S010`

每台电焊机上均设有铭牌，列出了该设备的关键技术参数，包括：
* **输入参数**：初级电压、相数、功率、发电机接法、转速等。
* **输出参数**：次级电压、空载电压、工作电压、电流调节范围、负载持续率等。
`证据：PPT_A000056_S018`

我国电焊机型号依据《电焊机型号编制方法》(GB/T 10249-2010)标准编制，采用汉语拼音字母和阿拉伯数字表示。型号编排中，第2、4项用阿拉伯数字表示，第3项用汉语拼音字母表示；若第3、4项不用时可空缺；改进序号按产品改进程序用阿拉伯数字连续编号。
`证据：PPT_A000056_S014`

#### 1.1.3 TIG焊程序控制系统逻辑
TIG焊的程序控制系统确保了焊接过程的自动化与安全，其核心循环流程包括：
1. **气流控制**：
   * **提前送气**：起弧前，必须用焊枪向起始焊点提前 **1.5～4s** 送气，以排除空气。
   * **滞后停气**：灭弧后应滞后 **5～15s** 停气，焊枪须待停气后才离开终焊处，防止高温金属氧化。
2. **电流控制**：
   * **升流**：焊接开始时，电流从较小的引弧电流逐渐上升到焊接电流。
   * **衰减**：焊接即将结束时，焊接电流应能自动衰减，直至电弧熄灭。
3. **引弧与稳弧**：接通焊接电源的同时，起动引弧装置。系统应自动控制引弧器、稳弧器的启动和停止。
4. **协调动作**：电弧引燃后进入焊接，焊枪的移动和焊丝的送进应同时协调进行。
5. **水冷同步**：使用水冷式焊枪时，送水与送气应同步进行。
`证据：PPT_A000145_S033`
`证据：PPT_A000145_S032`

#### 1.1.4 焊接辅助设备
* **焊接回转台**：将焊件绕垂直轴或倾斜轴回转的焊接变位机械。主要用于高度不大、具有环形焊缝焊件的焊接或封头的切割工作。其工作台一般处于水平或固定在某一倾角位置，并能保证以焊速回转。
`证据：PPT_A000056_S028`
* **变位机**：集翻转（或倾斜）和回转功能于一身的变位机械。翻转和回转分别由两根轴驱动，夹持焊件的工作台除能绕自身轴线回转外，还能绕另一根轴作倾斜或翻转。其核心功能是将焊件上各种位置的焊缝调整到水平或“船形”易施焊位置。
`证据：PPT_A000056_S029`

### 操作步骤
1. **检查设备连接**：对照设备组成图，确认电源、供气、供水、焊枪、工件电缆等12个部分连接正确且无松动。
2. **核对铭牌参数**：查看电焊机铭牌，确认输入电压、相数及输出电流范围是否符合现场电源条件。
3. **测试程序逻辑**：在不引弧的情况下，触发焊枪开关，观察气流是否提前送出，松开开关后观察气流是否延时停止。
4. **检查水冷系统**：若使用水冷焊枪，确认送水与送气同步启动，出水正常。

### 工艺要点
* **气流保护窗口**：严格遵循“提前1.5-4s送气，滞后5-15s停气”的原则，这是保证焊缝不被氧化的关键。
* **电流平滑过渡**：利用程序控制实现电流的平滑上升和衰减，避免引弧时电流过大烧穿工件，或收弧时电流突变产生弧坑。

### 常见错误
* **气流不同步**：起弧前未提前送气，导致焊缝起始端氧化；收弧后立即停气，导致弧坑氧化。
* **水冷不同步**：水冷焊枪送气但未送水，导致焊枪过热损坏。
* **参数误读**：未核对铭牌上的负载持续率，长时间超负荷使用导致焊机过热保护。

### 图/视频观察任务
* **观察任务1**：观看设备连接示意图（参考 `PPT_A000145_S010` 图片），重点识别焊枪气管、供气气管、出水管和开关线的区别与连接位置。
* **观察任务2**：观看TIG焊程序循环图（参考 `PPT_A000145_S033` 图片），重点观察手工TIG焊与自动TIG焊在电流升降和气流控制上的时序差异。

### 小结
本节介绍了钨极氩弧焊设备的硬件组成、电源特性及程序控制逻辑。掌握设备各部分的名称与功能是安全操作的前提，理解气流与电流的同步控制逻辑是保证焊接质量的基础。

## 1.2 送丝操作认知

### 学习目标
1. 能说明焊丝直径选择与送丝速度的关系。
2. 能根据喷嘴尺寸和电流大小选择保护气体流量。
3. 能区分外填丝法与内填丝法的定义及适用场景。
4. 能演示断续送丝法与连续送丝法的操作动作。

### 知识讲解

#### 1.2.1 焊丝直径与送丝速度
* **焊丝直径选择**：当板厚及接头间隙大时，焊丝直径应选大一些。
* **送丝速度影响因素**：焊丝的送丝速度与焊丝直径、焊接电流、焊接速度和接头间隙等因素有关。
  * 一般焊丝直径大时，送丝速度慢。
  * 焊接电流、焊接速度和接头间隙大时，送丝速度快。
`证据：PPT_A000145_S036`

#### 1.2.2 保护气体流量
TIG焊决定保护效果的主要因素包括：保护气流量、喷嘴尺寸、喷嘴与母材的距离、外来风等。
* **流量选择原则**：通常首先要考虑所需保护的范围、焊枪喷嘴尺寸以及所使用焊接电流的大小。
`证据：PPT_A000145_S036`

#### 1.2.3 焊丝填充方式
根据焊丝填充位置，主要分为外填丝法和内填丝法：
1. **外填丝法**：电弧在管壁外侧燃烧，焊丝从坡口一侧填加的操作方法。
2. **内填丝法**：电弧在管壁外侧燃烧，焊丝从坡口间隙伸入管内，向熔池送入的操作方法。
`证据：PPT_A000065_S046`
`证据：PPT_A000145_S082`
`证据：PPT_A000076_S036`

#### 1.2.4 外填丝法的具体操作
外填丝法又分为断续送丝法和连续送丝法：
1. **断续送丝法（点滴送入）**：
   * **操作**：靠手的反复送拉动作，将焊丝端头的熔滴送入熔池。
   * **细节**：熔化后将焊丝拉回，退出熔池，但**不离开气体保护区**。
   * **作用**：焊丝拉回时，靠电弧吹力将熔池表面的氧化膜除掉。
2. **连续送丝法**：
   * **操作**：将焊丝端头插入熔池，利用手指交替移动，连续送入焊丝。
   * **特点**：随着电弧向前不断移动，熔池逐渐形成。与自动焊送丝法类似，特点是电流大、焊速快、波纹细、成形美观。
   * **要求**：需手指连续稳定地交替移动焊丝，需要熟练的送丝技能。
`证据：PPT_A000065_S048`
`证据：PPT_A000076_S038`
`证据：PPT_A000145_S084`

### 操作步骤
1. **选择焊丝**：根据板厚和间隙选择合适直径的焊丝。
2. **设定气流**：根据喷嘴尺寸和电流大小设定保护气体流量。
3. **练习断续送丝**：
   * 引燃电弧，形成熔池。
   * 将焊丝送入熔池边缘，待熔滴过渡后迅速拉回。
   * 确保焊丝始终在氩气保护范围内。
4. **练习连续送丝**：
   * 保持焊丝端头在熔池中。
   * 手指有节奏地交替送进焊丝，配合焊枪移动。

### 工艺要点
* **保护范围**：无论采用何种送丝方式，焊丝端头严禁离开氩气保护区，否则会导致焊缝吸气产生气孔。
* **氧化膜清理**：断续送丝时，利用焊丝拉回时的电弧吹力清理熔池表面氧化膜，是保证熔合良好的关键。
* **手指稳定性**：连续送丝要求手指动作稳定，送丝速度需与焊接速度匹配，避免送丝过快导致未熔合或送丝过慢导致内凹。

### 常见错误
* **焊丝出保护区**：送丝时焊丝端头伸出喷嘴保护范围，导致焊缝氧化。
* **送丝不协调**：焊枪移动与送丝不同步，造成焊缝宽窄不均或余高过大/过小。
* **混淆内外填丝**：在需要内填丝的场景（如小直径管对接）错误使用外填丝，导致根部未焊透。

### 图/视频观察任务
* **观察任务1**：观看“送丝.flv”视频片段，重点观察外填丝法中焊丝从坡口一侧加入的动作，以及内填丝法中焊丝伸入管内的路径。
* **观察任务2**：对比观察断续送丝与连续送丝的手指动作差异，注意断续送丝时焊丝拉回是否清理了熔池表面。

### 小结
送丝操作是TIG焊的核心技能之一。正确选择焊丝直径和气流参数是基础，熟练掌握断续和连续送丝技巧则是保证焊缝成形和内部质量的关键。

## 1.3 焊前准备与基础操作

> **修订说明**：根据审核报告，原“盖面焊”章节顺序逻辑错误，且“焊前准备”和“引弧”被错误后置。此处将“焊前准备”、“非接触引弧”及“收弧操作”整合为基础操作章节，置于具体层焊（打底、填充、盖面）之前，符合技能习得逻辑。

### 学习目标
1. 能执行焊前工件清理与装配检查。
2. 能演示非接触引弧操作，确保起弧平稳。
3. 能执行正确的收弧操作，防止弧坑裂纹。

### 知识讲解

#### 1.3.1 焊前准备
焊前准备是保证焊接质量的前提，主要包括：
1. **工件清理**：清除坡口及两侧的油、锈、水、氧化膜等杂质，防止气孔和夹杂。
2. **装配检查**：检查坡口角度、钝边尺寸及间隙是否符合工艺要求。
3. **参数设定**：根据焊件材质、厚度及接头形式，设定合适的焊接电流、气体流量及钨极直径。
`证据：PPT_A000051_S021`
`证据：PPT_A000065_S018`
`证据：PPT_A000076_S018`

#### 1.3.2 非接触引弧
非接触引弧（高频引弧或脉冲引弧）是TIG焊常用的起弧方式，其特点是无需钨极接触工件，避免钨极污染和夹钨。
1. **操作要点**：
   * 焊枪对准起弧点，保持适当距离（通常2-4mm）。
   * 按下焊枪开关，高频高压击穿空气间隙，引燃电弧。
   * 引弧成功后，迅速将焊枪降至正常焊接高度。
2. **注意事项**：
   * 确保高频发生器工作正常，避免频繁引弧失败损坏钨极尖端。
   * 引弧点应选择在坡口起始端，便于后续操作。
`证据：PPT_A000051_S002`
`证据：PPT_A000055_S002`
`证据：PPT_A000056_S002`
`证据：PPT_A000057_S002`

#### 1.3.3 收弧操作
收弧操作不当易产生弧坑裂纹、缩孔等缺陷。
1. **操作要点**：
   * 焊接结束时，不要直接切断电源。
   * 利用电流衰减功能，逐渐减小焊接电流。
   * 在电流衰减过程中，继续填充焊丝，填满弧坑。
   * 待电弧熄灭后，保持焊枪停留片刻，确保滞后停气，保护高温金属。
2. **常见缺陷预防**：
   * **弧坑裂纹**：因冷却速度快、应力集中产生。通过填满弧坑和缓冷可预防。
   * **弧坑缩孔**：因金属收缩产生。通过充分填充焊丝可预防。
`证据：PPT_A000051_S016`
`证据：C000020`
`证据：PPT_A000145_S080`

### 操作步骤
1. **清理与装配**：使用钢丝刷清理坡口，检查间隙。
2. **设定参数**：根据工艺卡设定电流、气体流量。
3. **练习引弧**：在废板上练习非接触引弧，确保起弧迅速、平稳，无夹钨。
4. **练习收弧**：在废板上练习收弧，观察弧坑是否饱满，有无裂纹。

### 工艺要点
* **引弧距离**：保持2-4mm距离，过近易粘钨，过远引弧困难。
* **弧坑填充**：收弧时必须填满弧坑，这是防止裂纹的关键。
* **滞后停气**：收弧后务必等待气体停止后再移开焊枪。

### 常见错误
* **粘钨**：引弧时钨极接触工件，导致钨极污染。
* **弧坑裂纹**：收弧过快或未填满弧坑。
* **弧坑氧化**：收弧后立即移开焊枪，未执行滞后停气。

### 小结
焊前准备、引弧和收弧是TIG焊的基础操作。规范的焊前清理能减少缺陷，正确的引弧和收弧能保证焊缝起收头质量。
