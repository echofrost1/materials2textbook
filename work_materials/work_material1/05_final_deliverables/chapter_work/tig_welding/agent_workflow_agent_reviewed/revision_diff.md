+收弧操作不当易产生弧坑裂纹、缩孔等缺陷。
+1. **操作要点**：
+   * 焊接结束时，不要直接切断电源。

+   * 在电流衰减过程中，继续填充焊丝，填满弧坑。
+   * 待电弧熄灭后，保持焊枪停留片刻，确保滞后停气，保护高温金属。
+2. **常见缺陷预防**：
+   * **弧坑裂纹**：因冷却速度快、应力集中产生。通过填满弧坑和缓冷可预防。
+   * **弧坑缩孔**：因金属收缩产生。通过充分填充焊丝可预防。

 ### 工艺要点
-*   **熔合线控制**：盖面焊的核心是保证两侧熔合。必须通过摆动让电弧边缘“舔”到坡口边缘，否则极易产生未熔合。
-*   **直线度控制**：眼睛要盯着坡口边缘（冷边），而不是盯着熔池中心，这样才能焊直。
-*   **余高控制**：送丝量要与焊接速度匹配，避免余高过大或内凹。
+* **引弧距离**：保持2-4mm距离，过近易粘钨，过远引弧困难。
+* **弧坑填充**：收弧时必须填满弧坑，这是防止裂纹的关键。
+* **滞后停气**：收弧后务必等待气体停止后再移开焊枪。
 
 ### 常见错误
-*   **摆动幅度过小**：导致焊缝两侧未熔合，焊缝过窄。
-*   **送丝量不足**：导致焊缝内凹，余高差过大。
-*   **未清理氧化皮**：导致焊缝内部夹杂。
-*   **熔化冷边**：过度熔化坡口边缘，导致焊缝直线度变差，难以控制。
+* **粘钨**：引弧时钨极接触工件，导致钨极污染。
+* **弧坑裂纹**：收弧过快或未填满弧坑。
+* **弧坑氧化**：收弧后立即移开焊枪，未执行滞后停气。
